⚠️ 关于徽章（badges）：
  仓库使用 docs/badges/ 下的本地 SVG 文件，不依赖 img.shields.io。
  原因：shields.io 的 /badge/status-active 等前缀经常 404，
  且 GitHub 对外部动态 SVG 有时渲染不稳定。
  如需替换为 shields.io 动态版，自行修改 README 顶部的 <img src> 即可。

【语言切换说明】
--------------------------------------------------
仓库根目录有三个 README 文件：

  README.md       ← 默认入口（中英文双语，带语言切换链接）
  README_EN.md    ← 纯英文版（适合设为 GitHub Pages 主页）

GitHub 默认渲染 README.md，用户可点击顶部链接切换语言。
