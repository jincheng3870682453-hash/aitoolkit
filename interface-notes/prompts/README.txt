# Prompt 模板文件

这些 .txt 文件是对应 AI 在不同阶段使用的 Prompt 模板，
方便在不同 AI 平台（ChatGPT / Claude / 元宝 等）中复用。

- identify_interface.txt  → 模式A: 识别接口（增量）
- ask_user.txt            → 模式A: 询问用户确认
- fullscan_audit.txt      → 模式B: 全量扫描审计
- export_md.txt           → Phase 2: 导出 md
- generate_diagram.txt    → Phase 2: 生成 Mermaid 图
- ocr_merge.txt           → Phase 3: OCR 回流合并
