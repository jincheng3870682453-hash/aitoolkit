# Task
将 session.ai_notes 中的所有接口数据导出为 Markdown 格式的接口笔记。

# 输出格式要求

```markdown
# 接口笔记 — {项目名}

> 生成时间：{timestamp}
> 接口总数：{count}
> 生成方式：AI自动记录 + 用户手写批注

---

## {接口名} {风险emoji}

- **功能**：{description}
- **参数**：
  - `{param_name}` ({type}) - {description}
- **返回**：{return_type}
- **位置**：`{file_path}`
- **调用了**：{dependencies}
- **被调用**：{called_by}
- **风险**：{emoji} {level}

> 📝 手写区（打印后在此写你的理解/踩坑经验）：
>
> _______________________________________________________
>
> _______________________________________________________
>
> _______________________________________________________

---
```

# 规则
1. 每个接口之间用 `---` 分隔
2. 手写区留 3 行空白下划线，方便打印后手写
3. 风险等级用 emoji：🟢 低 / 🟡 中 / 🔴 高
4. 免责声明（仅模式B v1）：放在文件最顶部，醒目标注
5. 文件末尾附 Mermaid 图
